HOW TO PLAY
Royal Security Officer Training Manual
1
🔍 Inspect
Use your Security Tools to reveal hidden information about each caravan. Look for mismatches.
2
⚖️ Analyze
Compare the Claimed Sender with the Real Source. If values say MISMATCH, it's a threat!
3
✅ Decide
Choose your action: Allow safe caravans to boost Economy, or Deny & Report threats.
4
📝 Report
When reporting a threat, pick the correct threat type to earn max Wisdom.

THREAT TYPES GUIDE
🎣
Phishing / Spoofing Fake senders tricking you into revealing credentials.
🐴
Double Ext. Trojan Malware disguised as a safe file (e.g., .pdf.exe).
💣
Zip Bomb A tiny archive that crashes systems when expanded.
🎭
Social Eng. / Baiting Manipulating human fear or greed to gain access.
🔓
Unencrypted (MITM) HTTP traffic where data can be intercepted.
🔄
Fake Update Malicious payload disguised as a system patch.
DECISION GUIDE
🟢
ALLOW / OPEN Use when all tools show VALID / MATCH results. Boosts Economy +5%.
🔵
TRUST & VERIFY Sends to quarantine sandbox for a deep scan. Safe option when unsure.
🔴
DENY & REPORT Block the threat and file a report. Pick correct type for max Wisdom (+10).
🟠
IGNORE & BURN Destroy without reporting. Blocks threat but gives only +5 Wisdom.
